<task type="auto">
  <name>Eliminate useState from logo components</name>
  <files>components/sections/yield-section/token-logo.tsx, components/sections/yield-section/protocol-logo.tsx</files>
  <action>
    Remove useState from logo components to eliminate per-item state overhead.

    **For both token-logo.tsx and protocol-logo.tsx:**

    Current implementation uses useState for error handling:
    ```tsx
    const [hasError, setHasError] = useState(false);
    ```

    Replace with one of these approaches:

    **Option A: CSS-only fallback (Preferred)**
    Use onError handler that manipulates DOM directly or uses CSS:
    ```tsx
    export function TokenLogo({ src, symbol, className = '' }: TokenLogoProps) {
      const handleError = (e: React.SyntheticEvent<HTMLImageElement>) => {
        // Hide image, show fallback via CSS
        e.currentTarget.style.display = 'none';
        const fallback = e.currentTarget.nextElementSibling as HTMLElement;
        if (fallback) fallback.style.display = 'flex';
      };

      return (
        <div className={`relative ${className}`}>
          <img
            src={src}
            alt={`${symbol} logo`}
            onError={handleError}
            className="rounded-full ring-1 ring-theme-border"
          />
          <div className="absolute inset-0 hidden items-center justify-center bg-[#1a2225] rounded-full">
            <span className="font-mono text-xs text-[#708090]">
              {symbol?.charAt(0) || '?'}
            </span>
          </div>
        </div>
      );
    }
    ```

    **Option B: Use ref instead of state**
    ```tsx
    const imgRef = useRef<HTMLImageElement>(null);
    const handleError = () => {
      if (imgRef.current) {
        imgRef.current.style.display = 'none';
        // Show fallback sibling
      }
    };
    ```

    Test with broken logo URLs to verify fallback works.
  </action>
  <verify>
    - Logo components have no useState
    - Fallback still displays when logo fails to load
    - No TypeScript errors
  </verify>
  <done>
    Both logo components refactored to use CSS-only fallback with no useState.
  </done>

    **Implementation steps:**

    1. **Install react-window if not present:**
       Check package.json for `react-window`. If missing, add it.

    2. **Import FixedSizeList:**
       ```tsx
       import { FixedSizeList as List } from 'react-window';
       ```

    3. **Replace CommandList with virtualized list:**
       Current implementation:
       ```tsx
       <CommandList className="max-h-[300px]">
         <CommandGroup>
           {filteredItems.map((item) => (
             <CommandItem key={item.value} ...>
               {/* item content */}
             </CommandItem>
           ))}
         </CommandGroup>
       </CommandList>
       ```

       New implementation:
       ```tsx
       <CommandList className="max-h-[300px]">
         <CommandGroup>
           {filteredItems.length > 0 ? (
             <div style={{ height: `${Math.min(filteredItems.length * 48, 300)}px` }}>
               <List
                 height={300}
                 itemCount={filteredItems.length}
                 itemSize={48} // Adjust based on actual item height
                 width="100%"
                 itemData={{ items: filteredItems, selectedValues, onToggle }}
               >
                 {VirtualizedItem}
               </List>
             </div>
           ) : (
             <CommandEmpty>No items found.</CommandEmpty>
           )}
         </CommandGroup>
       </CommandList>
       ```

    4. **Create VirtualizedItem component:**
       ```tsx
       const VirtualizedItem = ({ index, style, data }: ListChildComponentProps) => {
         const { items, selectedValues, onToggle } = data;
         const item = items[index];
         const isSelected = selectedValues.includes(item.value);

         return (
           <div style={style} className="px-2 py-2">
             <CommandItem
               value={item.label}
               onSelect={() => onToggle(item.value)}
               className="flex items-center gap-2"
             >
               <Checkbox checked={isSelected} className="size-4" />
               {/* logos and other content */}
             </CommandItem>
           </div>
         );
       };
       ```

    5. **Handle empty state:**
       When filteredItems is empty, show CommandEmpty.

    6. **Adjust itemSize:**
       Measure actual item height including padding. Use browser DevTools if needed.
       Typical: py-2 (8px top + 8px bottom) + content (~32px) = ~48px

    - Keep keyboard navigation working
    - Ensure Checkbox state updates correctly
